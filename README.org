#+DESCRIPTION: Configurazione personale per workstation Arch Linux con Sway

Configurazione personale dei dotfiles per [[https://archlinux.org][Arch Linux]] basata su [[https://swaywm.org][Sway]] (compositor Wayland i3-compatible), [[https://codeberg.org/dnkl/foot][Foot]] (terminale) e [[https://i3wm.org/i3blocks][i3blocks]] (barra di stato).

* Dipendenze
Elenco dei pacchetti necessari:

** Sway e componenti
- =sway= – Compositor Wayland
- =swayidle= – Demone idle / sospensione
- =swaylock= – Schermata di blocco
- =swaync= – Centro notifiche
- =polkit-gnome= – Agenti di autenticazione

** Applicazioni
- =foot= – Terminale
- =rofi= – Avvio applicazioni (dmenu-style)
- =emacs= (con =emacsclient=) – Editor / productivity suite
- =i3blocks= – Moduli per la barra di stato
- =pavucontrol= – Mixer audio
- =darkman= – Toggle tema chiaro/scuro

** Utilità di sistema
- =pipewire= / =wireplumber= (=wpctl=) – Gestione audio
- =playerctl= – Controllo media (play/pause/next/prev)
- =brightnessctl= – Controllo luminosità
- =wlsyshot= – Screenshot
- =emojimenu= – Menu emoji
- =device-menu= – Menu dispositivi
- =dbus= (=dbus-update-activation-environment=)

** Font (consigliati)
- =otf-opendyslexic-nerd= – Font principale per terminale e sway

* Installazione
#+BEGIN_SRC sh
# Clona il repository
git clone https://github.com/NFVblog/workstation-dotfiles.git ~/.dotfiles

# Crea i symlink nella tua home
ln -sf ~/.dotfiles/sway    ~/.config/sway
ln -sf ~/.dotfiles/foot    ~/.config/foot
#+END_SRC

* Configurazioni

** Sway (~sway/)
Il compositor è configurato con un file principale =config= e moduli separati in =config.d/=:
- =keybind.conf= – Scorciatoie extra (screenshot, notifiche, emoji, dispositivi)
- =colors.conf= – Colori finestre e toggle darkman
- =media.conf= – Tasti multimediali (volume, luminosità, player)
- =gaps.conf= – Gaps tra le finestre (smart_gaps, inner/outer 5px)
- =gestures.conf= – Gesture touchpad per cambiare workspace
- =monitor.conf= – Configurazione monitor dual-display
- =app.conf= – Regole finestre per applicazioni (Thunderbird, GNOME Software, Syncthing)
- =applet.conf= – Applet di sistema
- =clipboard.conf= – Gestione appunti
- =keybord-layout.conf= – Layout tastiera
- =tablet.conf= – Configurazione tablet
- =touchscreen.conf= – Configurazione touchscreen

** Foot (~foot/)
Terminale minimalista con:
- Font OpenDyslexicM Nerd Font Mono (size 8)
- Tema chiaro/scuro con include dinamico (=themes/current.ini=)
- Cursor beam con blink
- Scrollback 10000 righe
- Padding centrato 5px
- DPI-aware

** XDG Desktop Portal (~xdg-desktop-portal/)
Portali configurati per:
- Screenshot e ScreenCast via =wlr=
- Temi via =darkman=
- Fallback GTK
