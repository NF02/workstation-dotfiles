Configurazione personale dei dotfiles per [[https://archlinux.org][Arch Linux]] basata su [[https://hyprland.org][Hyprland]] (compositor Wayland), [[https://ghostty.org][Ghostty]] (terminale) e [[https://github.com/Alexays/Waybar][Waybar]] (barra di stato).

* Dipendenze
Elenco dei pacchetti necessari:

** Hyprland e componenti
- =hyprland= – Compositor Wayland
- =hypridle= – Demone idle / sospensione
- =hyprlock= – Schermata di blocco
- =hyprsunset= – Filtro luce blu
- =hyprpolkitagent= – Permessi screencopy
- =xdg-desktop-portal-hyprland= – Portale per integrazione desktop
- =xdg-desktop-portal-gtk= – Portale GTK

** Applicazioni
- =ghostty= – Terminale
- =waybar= – Barra di stato
- =pcmanfm-qt= – File manager
- =hyprlauncher= – Avvio applicazioni (dmenu-style)
- =emacs= (con =emacsclient=) – Editor / productivity suite
- =swaync= – Centro notifiche
- =pavucontrol= – Mixer audio
- =iwd= (=iwctl=) – Gestione WiFi
- =obs-studio= – Permessi screencopy
- =hyprshot= – Screenshot (permessi concessi)

** Utilità di sistema
- =pipewire= / =wireplumber= (=wpctl=) – Gestione audio
- =playerctl= – Controllo media (play/pause/next/prev)
- =brightnessctl= – Controllo luminosità
- =darkman= – Toggle tema chiaro/scuro
- =qt6ct= – Theming Qt sotto Wayland
- =dbus= (=dbus-update-activation-environment=)

** Font (consigliati)
- =ttf-jetbrains-mono-nerd= – Font per Hyprlock
- =otf-opendyslexic-nerd= – Font per Ghostty (opzionale, modificabile nel config)

* Installazione
#+BEGIN_SRC sh
# Clona il repository
git clone https://github.com/NFVblog/workstation-dotfiles.git ~/.dotfiles

# Crea i symlink nella tua home
ln -sf ~/.dotfiles/ghostty  ~/.config/ghostty
ln -sf ~/.dotfiles/hypr     ~/.config/hypr
ln -sf ~/.dotfiles/waybar   ~/.config/waybar
#+END_SRC

* Configurazioni

** Hyprland (~hypr/~)
Il compositor è configurato via Lua (=hyprland.lua=), con moduli separati in =hypr/conf/=:
- =keybinding.lua= – Tutte le scorciatoie da tastiera
- =autorun.lua= – Variabili d'ambiente e avvio automatico (Waybar, SwayNC, portali)
- =window.lua= – Regole finestre, gaps, bordi, animazioni
- =permessi.lua= – Permessi screencopy (OBS, Hyprshot)
- =keybord-layout.lua= – Layout tastiera US intl., gesture touchpad

** Ghostty (~ghostty/~)
Terminale con:
- Tema chiaro/scuro automatico
- Font OpenDyslexicM Nerd Font Mono
- Shader cursore animato (=cursor_blaze.glsl=)
- Opacità sfondo 0.78 con blur

** Waybar (~waybar/~)
Barra di stato in alto con moduli: workspace, submode, titolo finestra, audio,
luminosità, CPU, RAM, rete, orario (Europe/Rome), vassoio di sistema.
